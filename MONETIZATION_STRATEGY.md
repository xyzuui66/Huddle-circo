## 1️⃣ **CREATOR PROGRAM (PRIMARY REVENUE - 70%)**

### **Cara Kerja:**

```
Viewer                          Creator
  ↓                               ↑
  Sends Gift (Rp 100K)
  ↓
  [Platform Transaction]
  ↓
  Commission cut 25% = Rp 25K (Platform)
  Net received 75% = Rp 75K (Creator)
  ↓
  Creator withdraw ke bank
  ↓
  Received at bank Rp 75K
```

### **Implementation:**

```typescript
// Gift Flow
1. User click "Kirim Gift"
2. Select gift (Rose Rp5K - Fire Rp2M)
3. Quantity & message
4. Click "Send"
5. Gift animation
6. Creator gets notification
7. Amount masuk ke wallet creator

// Withdrawal Flow
1. Creator go to Dashboard
2. Click "Withdrawal"
3. Enter amount (min Rp 50K)
4. Select payment method
5. Enter bank details
6. Submit
7. Admin verifikasi 1-24 jam
8. Transfer ke bank creator
```

### **Payment Methods (Xendit Integration):**

```
Supported:
- Bank Transfer (semua bank)
- E-wallet (OVO, GoPay, Dana)
- Kartu Kredit (cicilan)

Fees: 2.5% (dibayar creator)
Min: Rp 10,000
Max: Unlimited
Processing: 1-3 hari kerja
```

